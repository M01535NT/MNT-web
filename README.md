# MNT-web

Sitio personal de **Moisés Núñez** — Consultor Inmobiliario en Tijuana, BC.

Portafolio de servicios: compra, venta, renta y asesoría legal de propiedades.

## Stack

- **Vite** — desarrollo y build
- **TypeScript** — tipado estricto
- **CSS puro** con tokens MD3 (Material Design 3)
- **PWA** — offline-ready, instalable

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Producción en `dist/`.

## PWA

vite-plugin-pwa genera service worker y manifest automáticamente.
