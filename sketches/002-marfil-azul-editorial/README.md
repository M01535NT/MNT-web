## Variant: Marfil azul editorial

### Design stance
Confianza cálida y editorial, como consultor privado con criterio jurídico.

### Key choices
- Layout: portada clara con tarjeta lateral y secciones tipo documento premium.
- Typography: Libre Baskerville para títulos; Source Sans 3 para lectura limpia.
- Color: marfil, azul tinta y acento bronce.
- Interaction: tabs filtran servicios; CTA muestra toast.

### Trade-offs
- Strong at: legibilidad móvil, cercanía, sensación profesional humana.
- Weak at: menor impacto dramático que la versión oscura.

### Best for
- Perfil que quiere verse serio, accesible y confiable sin sentirse frío.
